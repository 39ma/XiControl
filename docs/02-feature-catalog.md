# Каталог функций — что можно положить в приложение

> 📌 **Исторический документ этапа планирования.** Актуальный статус — в
> [ROADMAP.md](../ROADMAP.md); ниже по строкам проставлены пометки «как вышло».

Легенда источника:
- 🟢 **WMI-native** — уникальная ценность, идёт через MIFS-прошивку.
- 🔵 **HW-event** — приходит WMI-событием (`HID_EVENT20`), нужен резидентный слушатель.
- ⚪ **Win32-generic** — стандартный Windows, к прошивке не относится, «бесплатно».

Оценка: **Core** — ядро продукта; **Nice** — приятное дополнение; **Later** — потом/по желанию; **Skip** — не тащить.

---

## Ядро (обязательно)

| Фича | Тип | cmd | Оценка | Комментарий |
|------|-----|-----|--------|-------------|
| Режим производительности | 🟢 | `0x08` | **Core** | Тихий / Турбо / Full-speed / Auto(Smart). Full-speed — только от сети. Auto только на `TM24*`. |
| Защита заряда 80/100% | 🟢 | `0x10` | **Core** | Бинарно. Порог зашит в прошивку: ≈80% на TM2424, на других моделях может отличаться (у MIControl — 70). Перед вкл — выкл (сброс EC). |
| Индикация в трее + меню | — | — | **Core** | Текущий режим/лимит, быстрое переключение. |
| OSD при переключении | — | — | **Core** | Как в CoreCharge — красиво и понятно (можно взять идею, не код). |
| Автозапуск при логине | — | — | **Core** | Task Scheduler, `/rl highest` (нужно для WMI Set — проверить). |

## Приятные дополнения

| Фича | Тип | cmd | Оценка | Комментарий |
|------|-----|-----|--------|-------------|
| Телеметрия: обороты вентиляторов | 🟢 | `0x0D` | ~~Nice~~ ❌ | На TM2424 `0xE0` — функции нет (проверено свипом). |
| Телеметрия: температура CPU | 🟢 | `0x16` | ~~Nice~~ ❌ | MIFS `0xE0`. ACPI-зона `\_TZ.TZ00` (WMI `MSAcpi_ThermalZoneTemperature`) тоже проверена (2026-07-13) — датчик непонятный: не реагирует на нагрузку/охлаждение/заряд в пределах тестов (но между днями значение меняется — не константа), а WMI-доступ к нему интермиттентный. Per-core/PCH — только MSR/драйвер. ⇒ **driver-free температуры нет.** |
| Тип БП (USB-C / DC) | 🟢 | `0x13` | ~~Nice~~ ❌ | На TM2424 `0xE0`. |
| Яркость подсветки клавиатуры | 🟢 | `0x12` | ~~Nice~~ ❌ | Через WMI нет (`0xE0`); уровень приходит **событием** `0x05` → сделан OSD. |
| Реакция на Fn-клавиши (OSD) | 🔵 | event | ✅ сделано | OSD: подсветка (4 уровня), микрофон, Fn-Lock, режимы. |
| Mi / AI кнопка → своё действие | 🔵 | event | ✅ сделано | Mi: клик/двойной/удержание (настраивается); AI: Copilot или своя программа. |
| Синхронизация mic-mute со звуком Windows | 🔵/⚪ | `0x0A`+audio | ✅ сделано | Core Audio mute + OSD по событию `0x21`. |
| Скрытый режим Эко | 🟢 | `0x08`=`0x0A` | ✅ сделано | Найден позже по EC-референсу, в каталоге не значился. |

## Позже / по желанию

| Фича | Тип | Оценка | Комментарий |
|------|-----|--------|-------------|
| Режим GPU (Hybrid/Discrete/UMA) | 🟢 `0x09` | **Later** | Мощно, но рискованно: обычно требует перелогина/перезагрузки, легко «уронить» вывод. Только после тестов. |
| Переключение частоты обновления | ⚪ | **Later** | Полезно на 120/165 Гц панелях, но это чистый `ChangeDisplaySettings`. |
| Вкл/выкл тачпада | ⚪ | **Later** | Обычный Windows-тумблер. |
| ICC цветовые профили | ⚪ | **Skip** | Нишевое, не про эту утилиту. |
| Произвольный % заряда (1–100) | ❌ EC | **Skip** | Осознанно отказались: требует WinRing0. Через WMI невозможно. |

## Мэппинг режимов — как вышло

Планировали 3 упрощённые кнопки, по факту показываем **все 5 режимов прошивки**
(Эко/Тихий/Авто/Турбо/Полная), а упрощение отдали пользователю: Эко и Полную
мощность можно скрыть галочками в Настройках (`EcoMode`/`FullSpeedMode`).

## Итоговый MVP — план vs факт

- **v0.1 (план) = v0.1 (факт):** трей + меню + OSD; режимы + заряд; автозапуск. ✅
- **v0.2 (план — телеметрия): не состоялся** — телеметрии на TM2424 нет (`0xE0`).
  Вместо неё v0.2–v0.5: SVG-иконки, анимации, жесты Mi, ремапы, Эко, Fn-Lock, китайский.
- **v0.3 (план — события): сделан раньше срока** и без службы — событий из admin-сессии
  оказалось достаточно (см. `03-architecture.md`).
