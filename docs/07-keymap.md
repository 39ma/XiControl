# Карта кодов клавиатуры — Xiaomi Book Pro 14 (TM2424)

Снято `map-keys.ps1` / `find-mibutton.ps1`, нажимая по одной клавише.
Событие: `EventDetail[1]`=code, `[2]`=value.

| code | value | Клавиша | Сейчас в системе | Что можно повесить |
|------|-------|---------|-------------------|--------------------|
| `0x21` | `0`=лампа горит (mute), `1`=лампа выкл | **Микрофон** (аппаратный тумблер с лампой) | ❌ не работает | синхр. системного mute + OSD |
| `0x01` | `0` | **Проекция экрана** | ❌ не работает | эмулировать Win+P |
| `0x1B` | `0` | **Настройки** | ❌ не работает | переключение заряда 80↔100 (одиночное событие, удержание не ловится) |
| `0x05` | `0`/`5`/`10`/`0x80` | **Подсветка клавиатуры** (off/5/10/Auto) | ✅ работает (EC) | OSD с уровнем |
| `0x23`+`0x24` | `1` (пара нажатие/отпускание) | **Нейропомощник (AI)** | ❓ не ясно | назначаемое действие |
| `0x25`+`0x26` | `1` (пара) | **Mi-кнопка** (2 тумблера) | ✅ наша утилита | клик/двойной/удержание |
| `0x07` | `0`/`1` = новое состояние | **Fn-Lock** (Fn+Esc, снято 2026-07-10) | ✅ EC переключает сам | OSD с замком. `1` = классические F1–F12 (мультимедиа выкл), `0` = мультимедийный ряд |

## Заметки
- **Микрофон инвертирован** относительно карты MIControl: лампа горит (замьючено) → `0`, лампа не горит → `1`.
  Клавиша аппаратно переключает лампу, но **в системе ничего не делает** — можно оживить (синхронизировать
  системный mute: `value==0` → mute, `value==1` → unmute).
- **Подсветка** уже работает на уровне EC; событие даёт уровень. Цикл значений: `0`(выкл) → `5` → `10` → `0x80`(Авто) → `0`.
  (Последние два — `0x0A`=10 и `0x80`=Авто; проверить, что `0x80` — именно авто-режим по датчику.)
- **Яркость экрана** — событий НЕ шлёт, Windows обрабатывает сама. Не трогаем.
- **AI-кнопка** (`0x23/0x24`) — как Mi, шлёт пару. Назначение действия на выбор владельца.

## Мёртвые клавиши для оживления (Tier 1)
1. Проекция (`0x01`) → Win+P.
2. Настройки (`0x1B`) → `ms-settings:`.
3. Микрофон (`0x21`) → системный mute (Core Audio) + OSD.
4. (опц.) Подсветка (`0x05`) → OSD с уровнем.
5. (опц.) AI (`0x23`) → назначаемое действие (Copilot / своя команда) — уточнить.
