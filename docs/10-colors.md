# Палитра XiControl

Фирменные цвета проекта. База — **Material Design**: приглушённые, «плоские», хорошо читаются
и в тёмной, и в светлой теме. Раздел «Иконки» — для рисования SVG; «UI» — цвета кода (WinForms).

Правило простое: **не вводи произвольные оттенки** — бери из таблиц ниже. Если нужного нет,
добавь сюда осознанно (ближайший Material-шейд), а не «на глаз».

### Правила иконок
- **Пара вкл/выкл**: «вкл» — цветная (синий/амбер); «выкл» — те же формы, синие → Blue Grey,
  **ripple/«шлейф» убирается**, «рука» `#FFCC80` остаётся (см. touchpad ↔ touchpad-off,
  touchscreen ↔ touchscreen-off).
- **Смысл цветом**: молния/заряд — амбер `#FFC107`; тач/экран — синий `#2196F3`;
  предупреждение — красный `#F44336`; эко/батарея — зелёный `#4CAF50`.
- **Оверлей-бейджи** заряда рисуются в полном кадре 128×128 на своём месте (угол) и
  накладываются 1:1 поверх базы — правится сама иконка, не код.
- **Svg.NET** молча игнорирует `<mask>`/`<clipPath>` — «дырки» только одним `<path>` с
  `fill-rule="evenodd"`. Трей-иконки — монохром `currentColor`, силуэты без тонких контуров.
