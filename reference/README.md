# reference/ — материалы-референсы

Складывай сюда всё, что пригодится при разработке и что не является исходниками нашего приложения:

- дампы вывода PowerShell-проб (см. `../docs/05-open-questions.md`);
- `probe-log.md` — журнал перебора WMI-команд (`cmd` → буфер → `OutData`);
- ETW/WMI-трейсы официального Xiaomi PC Manager (если будешь снимать);
- скриншоты UI, которые нравятся/не нравятся, как ориентир для дизайна;
- выдержки ACPI (DSDT) с описанием `PNP0C14`/`MIFS`, если решишь дизассемблировать;
- любые заметки.

Сами референс-проекты в репозиторий **не входят** (чужой код, см. `../docs/04-references.md`):
- Xiaomi-CoreCharge — вариант на WinRing0/EC;
- MI Control Utility — вариант на WMI (GPLv3; код не копировался, переносились только факты о протоколе).
