OSD Tray Icons — 4 монохромные иконки для системного трея (стиль Windows 11 / Fluent)

tray-perf-turbo.svg  — молния: турбо режим
tray-perf-quiet.svg  — листок: тихий режим (эко)
tray-perf-full.svg   — ракета: полный режим (максимальная мощность)
tray-settings.svg    — переключатели: настройки

Формат: сетка 24×24, контурный стиль, толщина линий 1.3–1.6, скруглённые
углы и концы. Силуэты повторяют цветной плоский набор — преемственность стиля.

Цвет: везде используется currentColor — иконка наследует цвет из CSS/приложения
и корректно инвертируется системой (белая на тёмной теме, чёрная на светлой).
Без CSS по умолчанию рендерится чёрной.

Рекомендуемые размеры: 16 / 20 / 24 px (проверены до 48 px).
Для .ico можно отрендерить в PNG 16/20/24/32 и собрать, напр.:
  rsvg-convert -w 16 tray-perf-turbo.svg > 16.png ... ; convert 16.png 20.png 24.png 32.png icon.ico
